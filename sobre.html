Sobre o projeto
Pomodorium

Este projeto foi desenvolvido como parte de um curso da Alura, focado em CSS e técnicas de layout responsivo. A base foi construída com HTML, CSS e JavaScript, e a partir dessa estrutura original, fiz personalizações visuais e funcionais para adaptar o sistema ao meu estilo e aprendizado.

O objetivo principal é aplicar a técnica Pomodoro de forma prática e acessível, promovendo ciclos de foco e descanso com um layout leve e agradável.

O código é aberto e faz parte do meu portfólio pessoal como desenvolvedor Full Stack.

O nome Pomodorium foi escolhido como uma junção criativa entre “Pomodoro” e um sufixo que lembra lugar ou reino. A ideia é que o site seja um pequeno universo de produtividade — um espaço focado, acolhedor e funcional onde o tempo é usado com intenção.
